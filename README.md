# DVLD---Driving-License-Management-System
A full desktop application for managing driving license operations, built from scratch using **C#, .NET, and SQL Server**.

---

## Overview

This system simulates a real-world driving license department workflow including:

* License issuance
* License renewal
* Replacement (lost/damaged)
* Detain & release licenses
* Test appointments management

---

## Architecture

The project follows a **3-Tier Architecture**:

* Presentation Layer (WinForms UI)
* Business Logic Layer (BLL)
* Data Access Layer (DAL)

---

## Technologies Used

* C#
